# Statement
 * ;으로 끝나야 한다.
 * 한 statement가 여러 줄에 걸쳐서 올 수도 있다.
 * {와 }를 이용해서 여러 statement를 block으로 묶을 수 있다.

## if Statement
## switch-case Statement
## for Statement

    for(initialize;condition;) {
        statements;
    }
    for(variable in object) {
        statements;
    }

## while Statement

    while(condition) {
        statements;
    }
    do {
        statements
    } while(condition);

## try catch

    try {
        throw err;
    } catch(err) {
    }
