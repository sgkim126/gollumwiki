# Object
 범용적인 데이터 타입.

 하지만 class는 존재하지 않는다.
 javascript는 prototype language로 class라는 틀을 이용해서 object를 찍어내는 것이 아니라, 다양한 object를 미리 만들어놓고, 그 object를 가져다 쓰거나, 복사해서 새로운 object를 만드는 방식을 이용한다.
 따라서 javascript에서는 생성자가 없다.
